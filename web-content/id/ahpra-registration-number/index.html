AHPRA Registration Number

URL

http://hl7.org.au/id/ahpra-registration-number

Type

identifiers namespace

Description

Australian Health Practitioner Regulation Agency Registration Number - allocated by AHPRA, practitioners are allocated one unique registration number for each profession in which they are registered. This number stays with the practitioner for life, even if they have periods when they are not registered. Practitioners registered in more than one profession have one registration number for each profession.

In OID-based systems 1.2.36.1.2001.1005.56 may be used.
