AHPRA Registrarion Number Mapping Guidance

Introduction

This page provides mapping guidance to the attributes of each AHPRA registration from a Practitioner information exchange (PIE) interoperability specification to elements in a profiled Practitioner resource. The table below demonstrates the logical decomposition of each data item to the lowest possible element in a Practitioner profile.

Mapping from PIE Find registration interoperability specification

This page provides mapping from the data items in interoperability specification of Australian Health Practitioner Regulation Agency (AHPRA)[1] [2].
