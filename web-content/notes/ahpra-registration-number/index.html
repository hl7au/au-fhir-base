AHPRA data guidance

This page provides detailed guidance on the representation of AHPRA-sourced data in a Practitioner resource. AHPRA data item names are taken from AHPRA's Practitioner information exchange (PIE) Interoperability specification: Find registration.

The table below matches the data items to the most appropriate elements in the Practitioner profile, or complex extensions AHPRA Profession Details and AHPRA Registration Details.

The following table maps AHPRA’s basic person data to Practitioner resource elements:

AHPRA person data
Data Item Element name
Name Edit Date name.period.start
Name Title name.prefix
Family Name name.family
Given Name name.given[1]
Middle Name name.given[2]
Profession Number identifier:AUAHPRARegistrationNumber

The following table maps AHPRA’s qualification data to an instance of the Practitioner resource qualification backbone element:

AHPRA qualification data
Data Item Element name
Qualification Title qualification.code.text
Awarding Institution qualification.issuer.resolve().name
Country Qualification Obtained qualification.issuer.resolve().address.country
Year of Qualification qualification.period.start
